# Weekly Search Console Review — Template

**Goal:** Move from index-checking to visibility-checking. Under 20 minutes per week.

---

## Review checklist

### 1. Top pages by impressions

Open **Performance → Pages**, sort by impressions, last 7 days.

- Which pages are earning the most impressions?
- Are the new reference pages (`/reference/...`) starting to appear?
- Is `/ai-compliance/` becoming the main compliance entry page?

**Note the top 5 pages and any changes from last week.**

### 2. Top queries by impressions

Open **Performance → Queries**, sort by impressions, last 7 days.

- What are people searching for when they see LatentMesh?
- Are there compliance-related queries pointing to the hub?
- Are there queries where position is improving week over week?

**Note the top 10 queries and any new entrants.**

### 3. Pages gaining new impressions

Filter for pages with impressions > 0, compare to previous period.

- Which pages gained impressions this week that didn't have them before?
- Are these the pages you expected?

### 4. Pages in positions 8–30 (striking distance)

Filter for average position between 8 and 30.

These are pages that are close to page 1 but not quite there. They are the highest-leverage targets for title/description/content improvements.

- Can you improve the title tag to better match the query?
- Can you add a better meta description?
- Can you add more internal links to the page?

**Note 2–3 striking-distance pages for attention this week.**

### 5. Country split

Check **Countries** tab for traffic from:
- United States
- Canada
- India
- Singapore
- EU countries (Germany, Netherlands, France, etc.)

Note any unexpected country patterns.

---

## Weekly output format

```markdown
## Week of [date]

### Winners
- [page] — [impressions], [avg position], [notable query]

### Pages to improve
- [page] — [reason: bad position, low CTR, weak title, etc.]

### Queries to watch
- [query] — [position], [page it maps to], [opportunity]

### One next action
- [specific action: update title, add internal links, improve content, etc.]
```

---

## Process notes

- Review every Monday or Tuesday
- Keep it under 20 minutes
- Focus on trends, not individual fluctuations
- Decisions should come from impression and query data, not guesswork
- Do not chase daily position changes — look at 7-day or 28-day trends
- Save each weekly note in this directory for reference
